# Исследование зависимости ширины спектра ЛЧМ сигнала от параметра девиации частоты

### Постановка задачи

ЛЧМ-сигнал описывается выражением:

$$
s(t) = \cos\left( \phi_0 + 2\pi \left( f_0 t + \frac{\Delta f}{2T} t^2 \right) \right)
$$

где:
- $\( f_0 \)$ — начальная частота,
- $\( \Delta f = F - f_0 \)$ — девиация частоты,
- $\( T \)$ — длительность сигнала,
- $\( \phi_0 \)$ — начальная фаза.

Теоретически ширина спектра ЛЧМ-сигнала приблизительно равна девиации частоты \(\Delta f\). На практике, из-за дискретизации, оконного эффекта, шума и метода оценки границ, наблюдаемая ширина спектра может отличаться.

## Алгоритм работы

1. Генерация ЛЧМ-сигнала с параметрами:
   - $f0$ — начальная частота,
   - $F$ — конечная частота (девиация),
   - $T$ — длительность,
   - $fd$ — частота дискретизации,
   - $phi0$ — начальная фаза.

2. Добавление шума
3. Спектральный анализ: Строится амплитудный спектр $\(|S(f)|\)$
4. Оценка ширины спектра: Определяются левая и правая границы, где спектр превышает порог, $\( \Delta f_{\text{exp}} = f_{\text{right}} - f_{\text{left}} \)$
5. Построение зависимости:
   - Параметр $F$ изменяется от $Fmin$ до $Fmax$ с шагом $Fstep$,
   - Для каждого $F$ вычисляется $\( \Delta f_{\text{exp}} \)$,
   - Строится график $\( \Delta f_{\text{exp}}(F) \)$.


На выходе программа строит три графика:

1. ЛЧМ-сигнал во временной области.
2. Амплитудный спектр зашумлённого сигнала с пороговой линией.
3. Зависимость ширины спектра от девиации частоты — основной результат исследования.
